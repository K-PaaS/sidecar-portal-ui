## 시작하기
K-PaaS 사이드카 WEB UI가 수행하는 애플리케이션 관리 작업은 다음과 같습니다.
- 사이드카 자원 관리
- 권한 관리
- 사용자 관리

#### 사이드카 WEB UI 빌드 방법
K-PaaS 사이드카  WEB UI 소스 코드를 활용하여 로컬 환경에서 빌드가 필요한 경우 다음 명령어를 입력합니다.
```
$ gradle build
```

<br>
